PDF Compression

To run the addon I suggest to have at least Ghostscript Version 9.05.

You can run on your server/vm: "sudo apt-get install ghostscript" this will upgrade your Ghostscript to the newest version.

The configuration is done under Settings/Attachment. Here can you select the different Ghostscript compression states. For more information about the compression states go to:

http://ghostscript.com/doc/current/Ps2pdf.htm

By Default the compression is not active:

you have to give the ir.attachment create function the context ('compress')

Example:

env['ir.attachment'].with_context(compress='compress').create(attachment_value)

If you want all pdf documents compresses by default just remove the line

if self._context.get('compress'):

from the create mehtod in the ir_attachment.py

Ghostscript Compression States

/screen selects low-resolution output similar to the Acrobat Distiller "Screen Optimized" setting.

/ebook selects medium-resolution output similar to the Acrobat Distiller "eBook" setting.

/printer selects output similar to the Acrobat Distiller "Print Optimized" setting.

/prepress selects output similar to Acrobat Distiller "Prepress Optimized" setting.

/default selects output intended to be useful across a wide variety of uses, possibly at the expense of a larger output file.
